# Train Consist Management App

## Project Description
This application is designed to facilitate the management of train consists. It allows users to view, add, and modify details about train cars, ensuring optimal organization and efficiency in train operations.

## Use Case 1 (UC1)
**Title:** Adding a New Train Car

**Description:** The user can add a new train car by entering details such as car number, type, and capacity. This information is validated before being added to the database.

**Actors:** Train Manager

**Preconditions:** The user must be logged into the application.

**Postconditions:** A new train car entry is created in the system, and the user receives confirmation of the successful addition.
